# Lira Tracker

A high-performance financial tool built with AndroLua to instantly convert old to new Lira by removing 2 zeros, while tracking real-time rates for global currencies, gold, and crypto assets.

## Key Features

* **Smart Conversion:** Instantly convert Old Lira to New Lira with support for 6 global currencies using live exchange rates.
* **Gold Market Watch:** Stay updated with accurate prices for various gold carats.
* **Crypto Monitor:** Track top cryptocurrency prices in both USD and Lira.
* **Accessibility First:** Fully optimized for screen readers, specifically tested with **Jieshuo** for a seamless experience.
* **Offline Mode:** Works without an internet connection by securely caching the latest updated data locally.
* **Lightweight & Fast:** Minimalist design for high performance on all Android devices.

## How to Use

1. Type the value in Old Lira.
2. View the redenominated value and its equivalent in global currencies.
3. Easily navigate between Currency, Gold, and Crypto sections.

## Technical Overview

* **Framework:** AndroLua and other old Android libraries.
* **Data Source:** Real-time financial APIs like SP Today.
* **Target Audience:** Users transitioning to the new currency system and financial traders.
